What are Server Actions?

React 서버 액션은 서버에서 직접 비동기 코드를 실행할 수 있게 합니다. 데이터를 변경하기 위해 API 엔드포인트를 생성할 필요가 없습니다. 대신에, 서버에서 실행되는 비동기 함수를 작성하고 클라이언트 또는 서버 컴포넌트에서 호출할 수 있습니다.

웹 애플리케이션은 다양한 위협에 취약할 수 있기 때문에 보안은 최우선 과제입니다. 이것이 서버 액션이 필요한 이유입니다. 서버 액션은 다양한 유형의 공격으로부터 보호하는 효과적인 보안 솔루션을 제공하여 데이터를 보호하고 인가된 액세스를 보장합니다. 서버 액션은 POST 요청, 암호화된 클로저, 엄격한 입력 검사, 오류 메시지 해싱 및 호스트 제한과 같은 기술을 통해 이를 달성하며, 이 모든 것은 앱의 안전성을 크게 향상시킵니다.

Using forms with Server Actions

리액트에서는
요소의 action 속성을 사용하여 액션을 호출할 수 있습니다. 액션은 자동으로 캡처된 데이터를 포함하는 원시 FormData 객체를 받게 됩니다. 예를 들어: 
					
						// Server Component
						export default function Page() {
							// Action
							async function create(formData: FormData) {
								'use server';

								// Logic to mutate data...
							}

							// Invoke the action using the "action" attribute
							return ...;
						}
